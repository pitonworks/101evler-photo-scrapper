# Workflow

## Geliştirme Akışı
1. `npm start` ile sunucuyu başlat
2. `http://localhost:3000` üzerinden arayüze eriş
3. Değişiklik yap → sunucuyu yeniden başlat

## Commit Conventions
- `feat: açıklama` — yeni özellik
- `fix: açıklama` — hata düzeltme
- `refactor: açıklama` — yeniden yapılandırma
- `docs: açıklama` — dokümantasyon
- `chore: açıklama` — bakım

## Test Etme
1. Photo Scraper: 101evler linki gir → fotoğraflar indirildi mi?
2. Listing Transfer: 101evler linki + gelgezgor bilgileri → ilan oluştu mu?
